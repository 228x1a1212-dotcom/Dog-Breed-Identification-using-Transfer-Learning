About the Project

Data Preparation: The dataset contains thousands of labeled dog images, cleaned and preprocessed for better accuracy.

Model Building: Using transfer learning with the VGG19 model, we fine-tuned the layers for breed classification.

Model Testing & Evaluation: The model achieved impressive accuracy after extensive validation on unseen images.

Deployment: The trained model is deployed via a Flask web interface for real-time dog breed predictions.
